# 🏡 Wanderlust - Airbnb Inspired Travel Booking Platform

Wanderlust is a full-stack travel booking platform inspired by Airbnb where users can discover destinations, create listings, upload images, and share reviews.

---

## 🌐 Features

### 🔐 Authentication
- User Registration & Login
- Secure Authentication
- Session Management

### 🏠 Listings
- Create Listings
- Edit Listings
- Delete Listings
- View Listing Details

### ⭐ Reviews
- Add Reviews
- Delete Reviews
- Rating System

### 📷 Media
- Image Upload Support
- Cloud Storage Integration

### 📱 User Experience
- Responsive Design
- Clean UI
- Mobile Friendly

---
